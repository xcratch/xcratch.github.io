# Xcratchとは

Xcratch (エクスクラッチ)とは、[MITのスクラッチ3](https://scratch.mit.edu/)をユーザーがつくった拡張機能で簡単に遊べるように変更したモッド・アプリケーションです。

## 特徴
### 必要に応じた拡張機能の読み込み

ユーザーがつくった拡張機能をウェブからインストールして使う事ができます。拡張されたブロックを使ったプロジェクトを Xcratch で開くと、使っている拡張機能を自動的にインストールして開きます。

拡張機能は以下のようなタグで分類され、目的に応じて必要な拡張機能を選んだり検索したりして利用できます。

### 先読み済みの拡張機能

Xcratch エディターには、あらかじめいくつかの便利な拡張機能が先読み (プリロード) されています。これにより、インターネットに接続していない状態でも、これらの拡張機能を利用できます。

### URL で拡張機能をあらかじめ読み込む

エディターの URL に `extension` クエリパラメータを追加することで、プロジェクトを開始する前に拡張機能をあらかじめ読み込めます。

https://xcratch.github.io/editor/?extension=<拡張機能URL>

パラメータを繰り返すことで、複数の拡張機能を同時に読み込むこともできます。

### TurboWarp 拡張機能のサポート

Xcratch は [TurboWarp](https://turbowarp.org/) の拡張機能に対応しています。TurboWarp 向けに作られた拡張機能を Xcratch で読み込んで利用できます。

### URLからプロジェクトを開く

Xcratchは、インターネット上で公開されているプロジェクトを直接開くことができます。
プロジェクトを直接開くには2つの方法があります。

エディタで `#` を指定して開く

https://xcratch.github.io/editor/#<プロジェクトURL>`。

または、クエリパラメータに `project` というラベルを追加します。

https://xcratch.github.io/editor/?project=<プロジェクトURL>`。

### Web Bluetooth API 対応

'Web Bluetooth API' に対応したブラウザ (Chrome や Edge 、 iPad では [Scrub](https://apps.apple.com/jp/app/scrub-web-browser/id1569777095) など) で開くと、micro:bit や他の BLE 接続のデバイスを [Scratch Link](https://scratch.mit.edu/microbit) 無しで遊ぶことができます。

### オフラインで利用可能

Xcratch エディターは PWA (Progressive Web Apps) になっているので、一度開けばそれ以降はインターネット通信ができない状態でも利用できます。
(拡張機能によってはインターネットの接続が必要な場合があります)

PWAなので、必要に応じてローカルマシンにインストールすることもできます。

### ローカル　バックパック

Xcratch にはバックパック機能が搭載されており、スプライトやコスチューム、サウンド、コードブロックなどをバックパックに入れて、他のプロジェクトで簡単に再利用できます。

バックパックはブラウザのローカルストレージを利用しているので、インターネットに接続されていない状態でもバックパックのデータを利用できます。

### ブラウザにプロジェクトを保存

Xcratch はアカウントやサーバーを使わずに、プロジェクトをブラウザ内に保存できます。この機能は通常のエディター (https://xcratch.github.io/editor/) で利用できます。

- **自動保存**: 編集中は1分ごとに自動的に保存され、次にエディターを開いたときは前回のプロジェクトから自動的に再開されます。
- **プロジェクト一覧**: 「ファイル > プロジェクト一覧から開く」を選ぶと、このブラウザに保存されているすべてのプロジェクトをサムネイル・名前・更新日時とともに確認できます。一覧からそのまま名前の変更・コピー・削除・コメントの記入ができます。
- **履歴**: 保存するたびにそのプロジェクトの履歴にスナップショット（サムネイル付き）が追加されるので、以前のバージョンを見返したり、いつでも復元したりできます。古い履歴は容量を抑えるため少しずつ間引かれますが、直近の履歴は細かく残ります。
- **複数タブでの編集**: 同じプロジェクトを2つのタブで編集していた場合、保存時に競合を検出し、自分の変更を残すか、それを破棄して最新の保存内容を読み込み直すかを選べます。

プロジェクトはブラウザの IndexedDB に保存されるため、このブラウザとこの端末に紐づいています。別の場所に持ち出したり長期的に保管したりしたい場合は、「ファイル > コンピューターに保存する」で `.sb3` ファイルとしてダウンロードしてください。

### ブロックを画像にエクスポート

ブロックエリアで右クリックすると、選択したブロックやすべてのブロックを SVG または PNG 形式の画像としてエクスポートできます。作成したブロックのスクリーンショットを簡単に保存して共有できます。

### エディターで変数やリストの値を書き換える

「変数の値」ブロックや「リストの◯番目の値」ブロックを右クリックして「エディターで値を書き換える」を選ぶと、画面内にテキストエディターが開き、変数やリストのアイテムの値を直接書き換えられます。複数行のテキストも編集できるので、改行を含む長い文章を変数に入れるときに便利です。

### エディターでリスト全体を編集する

リスト全体を表示する「リスト名」ブロック（リスト全体を返す丸いブロック）を右クリックして「リストを編集...」を選ぶと、リストエディターのモーダルが開きます。

一覧画面では、ドラッグハンドルや ↑ / ↓ ボタンでアイテムを並び替えたり、「+ アイテムを追加」で新しいアイテムを追加したり、× ボタンで削除したりできます。これらの変更はリストに即時反映されます。「元に戻す」ボタンを押すと、モーダルを閉じずに、開いたときの状態にリストを戻せるので、編集中の間違いをいつでも取り消せます。

各行を選ぶと、長文や複数行を編集できる大きなテキストエリアが開きます。最大化ボタンを押すと、モーダルが画面いっぱいに広がるので、長い段落を編集するときに便利です。← 戻るボタン・× ボタン・モーダル外のクリックで編集を確定し、「キャンセル」で編集を破棄します。

### Scratch Foundation版からのUIの改善

- 全角文字で入力した数字を数値として利用できる
- 日本語の算術演算子に見やすいシンボル(＋, －, ✕, ／)を使っている
- 「言う」「考える」や「=」判定に改行(\n)やタブ(\t)文字を使える
- コードの「きれいにする」で横方向の整列やアニメーションなどが改善されている
- コメントの位置が変わったりするコメントのバグが修正されている
- ブロックパレットの右端にかくれた部分が見える
- 独自サーバーでも Backpack を使える
- HTML埋め込みで利用するときのエディタを開く機能や、フルスクリーンモードのサポート
